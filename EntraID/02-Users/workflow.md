\# Users Management in Microsoft Entra ID



\## Objective

To manage and explore user accounts within Microsoft Entra ID for centralized identity management.



\## Environment

\- Platform: Microsoft Entra ID

\- Domain: DomainExpansion874.onmicrosoft.com



\## Steps Performed

\- Navigated to the Users section in Entra ID

\- Created and managed user accounts

\- Reviewed user details including usernames and properties

\- Centralized user management simplifies administration in cloud environments
