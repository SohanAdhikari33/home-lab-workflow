# Groups Management in Microsoft Entra ID

## Objective
To manage user access and permissions using groups in Microsoft Entra ID.

## Environment
- Platform: Microsoft Entra ID
- Domain: DomainExpansion874.onmicrosoft.com

## Steps Performed
