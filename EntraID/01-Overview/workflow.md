\# Microsoft Entra ID - Overview



\## Objective

To explore and understand cloud-based identity and access management using Microsoft Entra ID in a home lab environment.



\## Environment

\- Platform: Microsoft Entra ID (Azure Active Directory)

\- Domain: DomainExpansion874.onmicrosoft.com

\- Integration: Independent cloud identity environment



\## Overview

This setup represents a cloud identity system where users, groups, devices, and access policies are managed centrally through Microsoft Entra ID.
