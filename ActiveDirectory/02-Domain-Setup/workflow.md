\# Domain Setup - Active Directory



\## Objective

To configure a domain environment by setting up Active Directory Domain Services and promoting the server to a Domain Controller.



\## Environment

\- Domain: DomainExpansion.com

\- Role: Domain Controller

\- Service: Active Directory Domain Services (AD DS)

\- System: Windows Server



\## Steps Performed

\- Installed Active Directory Domain Services (AD DS)

\- Promoted the server to a Domain Controller

\- Created a new domain: DomainExpansion.com

\- Verified domain configuration
