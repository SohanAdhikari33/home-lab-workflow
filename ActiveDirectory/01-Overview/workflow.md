\# Active Directory - Overview



\## Objective

To set up and demonstrate a working Active Directory environment in a home lab to simulate a real world IT infrastructure.



\## Environment

\- Domain: DomainExpansion.com

\- Role: Domain Controller

\- Services: Active Directory Domain Services (AD DS)

\- System: Windows Server



\## Overview

This environment represents a centralized identity and access management system where users, devices, and policies are managed within a domain.
