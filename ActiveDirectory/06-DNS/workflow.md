\# Domain Join



\## Objective

To demonstrate domain-based authentication by logging into a client system using Active Directory credentials.



\## Environment

\- Domain: DomainExpansion.com

\- Client System: Windows 10/11

\- Domain Controller: Windows Server



\## Steps Performed

\- Connected client system to the domain (DomainExpansion.com)

\- Used domain credentials to log in

\- Verified authentication through domain login

Successfully authenticated a user through the Active Directory domain.



\## Key Learnings

\- Domain login enables centralized authentication

\- Users can access systems using domain credentials

\- Active Directory controls authentication across the network
