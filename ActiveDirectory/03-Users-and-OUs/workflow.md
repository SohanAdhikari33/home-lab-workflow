\# Users, Organizational Units (OUs), and Groups



\## Objective

To organize users and manage access efficiently using Organizational Units and Security Groups in Active Directory.



\## Environment

\- Domain: DomainExpansion.com

\- Tool: Active Directory Users and Computers



\## Steps Performed

\- Created Organizational Units (OUs): Finance, IT, Sales, HR

\- Created users and assigned them to respective departments

\- Created Security Groups for each department

\- Added users to their respective groups for access control
