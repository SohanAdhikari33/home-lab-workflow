# Microsoft 365 – Overview

## Objective
To explore Microsoft 365 collaboration tools and understand user, license, and service management in a lab environment.

## Environment
- Platform: Microsoft 365
- Domain: DomainExpansion874.onmicrosoft.com
- Subscription: E3/E5 trial (15 days)
- Integration: Connected with Microsoft Entra ID

## Overview
This Microsoft 365 environment provides cloud-based productivity and collaboration tools including Exchange Online, Teams, OneDrive, and SharePoint.

It allows administrators to manage users, assign licenses, and configure services while providing real-time collaboration for end users.
