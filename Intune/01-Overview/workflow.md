\# Microsoft Intune - Overview



\## Objective

To explore and understand device management and security using Microsoft Intune in a home lab environment.



\## Environment

\- Platform: Microsoft Intune

\- Domain: DomainExpansion874.onmicrosoft.com

\- Integration: Connected with Microsoft Entra ID



\## Overview

This setup represents a cloud-based device management system where devices, applications, and security policies are managed centrally using Microsoft Intune.

\## Key Learnings

\- Microsoft Intune enables centralized device management

\- It integrates with Entra ID for identity-based control

\- It allows administrators to manage security and compliance remotely
