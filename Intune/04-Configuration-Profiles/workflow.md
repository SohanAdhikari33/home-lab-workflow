\# Microsoft Intune - Configuration Profiles



\## Objective

To demonstrate the creation and management of configuration profiles in Microsoft Intune, enforcing device settings and security.



\## Environment

\- Platform: Microsoft Intune

\- Domain: DomainExpansion874.onmicrosoft.com

\- Integration: Connected with Microsoft Entra ID



\## Steps Performed

\- Navigated to Configuration Profiles section in Intune

\- Reviewed the list of configuration profiles created

\- Opened individual profiles to verify applied settings such as password requirements and device restrictions

\- Ensured policies were targeted to the enrolled devices

\## Key Learnings

\- Configuration profiles enforce device settings centrally

\- Profiles can include password policies, encryption, restrictions, and other security measures

\- Targeting profiles to devices ensures consistent compliance across the environment
