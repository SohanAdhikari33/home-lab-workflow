\# Microsoft Intune - Application Management



\## Objective

To demonstrate deployment and management of applications on enrolled devices using Microsoft Intune.



\## Environment

\- Platform: Microsoft Intune

\- Domain: DomainExpansion874.onmicrosoft.com

\- Integration: Connected with Microsoft Entra ID



\## Steps Performed

\- Navigated to Application Management section in Intune

\- Reviewed the list of available and deployed applications

\- Assigned applications to enrolled devices

\- Verified the application installation on devices
