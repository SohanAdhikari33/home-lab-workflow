# IT Support / Cloud Home Lab Project

## Overview

This repository documents my hands-on IT support and cloud environment setup using Microsoft technologies.

The goal of this project was to simulate a real-world enterprise IT environment, covering identity management, device management, security, and ticketing workflows.

## Technologies Used

- Active Directory
- Microsoft Entra ID
- Microsoft Intune
- Microsoft 365 (Exchange, Teams, Outlook)
- Microsoft Defender
- Microsoft Purview
- Jira (Ticketing System)

## Project Structure

- Active Directory → Domain setup, users, groups, GPOs
- Entra ID → Cloud identity and user management
- Intune → Device management and policies
- Microsoft 365 → Collaboration tools and admin center
- Microsoft Defender → Security monitoring and alerts
- Microsoft Purview → Compliance and data protection
- Ticketing System → IT support workflow simulation

## Key Skills Demonstrated

- Identity and Access Management (IAM)
- Device Management (MDM)
- Security Monitoring and Incident Handling
- Data Protection and Compliance
- IT Support Ticket Lifecycle Management
- Documentation and Troubleshooting

## Outcome

This project helped me build practical, hands-on experience in IT support and cloud administration, while also improving my ability to document and present technical workflows.
